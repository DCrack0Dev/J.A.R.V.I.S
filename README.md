# J.A.R.V.I.S. (Just A Rather Very Intelligent System)

Personal Operations Interface for schedule management, job hunting, and GitHub intelligence.

## Features
- **Voice-Controlled HUD**: Interactive dashboard with real-time schedule tracking.
- **Job Hunter**: AI-powered job board scraper and application assistant.
- **GitHub Intelligence**: Automated repo health monitoring and README generation.
- **Schedule Architect**: AI-driven daily routine optimization.

## Tech Stack
- **Frontend**: Next.js / Vite
- **Backend**: NestJS
- **Database**: PostgreSQL / Prisma
- **AI**: OpenRouter (Claude-3.5-Sonnet)
